# Component Library Todo-List

## 1. Primitives & Layout (The Foundations)

These components are used to position the others.

- [x] **Box** (The base polymorphic container)
- [x] **Stack** (Vertical/horizontal Flexbox layout with managed `gap`)
- [x] **Grid** (Wrapper for CSS Grid)
- [x] **Container** (Container with `max-width` to center content)
- [x] **Divider** (Horizontal/vertical separator line)

## 2. Inputs (User Interaction)

The heart of the functional interface.

- [ ] **Button** (High Priority ⭐️)
- [ ] **IconButton** (Button containing only an icon, often round)
- [ ] **Input / TextField** (Text field, with variants for icons or label)
- [ ] **Textarea** (Multiline input)
- [ ] **Checkbox** (Multiple selection)
- [ ] **RadioGroup** (Single selection)
- [ ] **Switch** (Toggle on/off)
- [ ] **Select** (Native or custom dropdown)

## 3. Data Display (Show Content)

- [x] **Typography (Heading / Text)**
- [ ] **Card** (Container with border/shadow and background to group info)
- [ ] **Badge** (Small numeric or status indicator, e.g. above an icon)
- [ ] **Avatar** (User profile image or initials)
- [ ] **Chip / Tag** (Compact labels, e.g. categories)
- [ ] **List** (Styled ordered/unordered list)
- [ ] **Table** (Tabular data display - _optional at first_)

## 4. Feedback (System Response)

- [ ] **Alert** (Static warning message: success, error, warning)
- [ ] **Skeleton** (Loading placeholder that simulates content shape)
- [ ] **CircularProgress / Spinner** (Loading indicator)
- [ ] **Toast / Snackbar** (Temporary floating notification)

## 5. Navigation & Surfaces (App Structure)

- [ ] **Link** (Styled anchor tag)
- [ ] **Breadcrumbs** (Navigation path)
- [ ] **Dialog / Modal** (Modal window that blocks interaction below)
- [ ] **Drawer** (Sliding side panel)
