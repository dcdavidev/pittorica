# Grid

Responsive 12-column grid system. Use column span utilities for layout control.

## Column Span Utility

- `.picto-grid-col-6` sets `--picto-grid-col-span: 6;`

## Usage

```html
<div class="picto-grid">
  <div class="picto-grid-col-6">...</div>
  <div class="picto-grid-col-6">...</div>
</div>
```
