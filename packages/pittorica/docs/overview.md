# Pittorica Overview

Pittorica is a modern, CSS-first design system inspired by Material Design 3, TailwindCSS, and MUI/Joy. It provides:

- **Design tokens** for colors, typography, spacing, radius, elevation, and motion
- **Responsive and themeable** via CSS custom properties
- **Utility classes and base components** for rapid development
- **SSR-friendly** and zero-runtime for maximum performance

## Key Features

- Dynamic color palettes with `color-mix()`
- Responsive typography and spacing in `rem` with `clamp()`
- Semantic tokens for easy theming and overrides
- Designed for React wrappers and any frontend framework
