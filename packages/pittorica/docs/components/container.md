# Container

Responsive wrapper for page content. Provides max-width, horizontal padding, and adaptive breakpoints.

## Variants

- `.picto-container-xs` – max-width: 24rem
- `.picto-container-sm` – max-width: 36rem
- `.picto-container-md` – max-width: 48rem
- `.picto-container-lg` – max-width: 64rem
- `.picto-container-xl` – max-width: 75rem
- `.picto-container-fluid` – full width, no max

## Usage

```html
<div class="picto-container picto-container-md">...</div>
<div class="picto-container picto-container-fluid">...</div>
```
