# Pittorica Buttons

Material Design 3 inspired button components. Includes:

- **Elevated**: `.picto-btn.picto-btn-elevated`
- **Filled**: `.picto-btn.picto-btn-filled`
- **Tonal**: `.picto-btn.picto-btn-tonal`
- **Outlined**: `.picto-btn.picto-btn-outlined`
- **Text**: `.picto-btn.picto-btn-text`
- **Icon**: `.picto-btn.picto-btn-icon`

## Usage

```html
<button class="picto-btn picto-btn-filled">Filled</button>
<button class="picto-btn picto-btn-elevated">Elevated</button>
<button class="picto-btn picto-btn-tonal">Tonal</button>
<button class="picto-btn picto-btn-outlined">Outlined</button>
<button class="picto-btn picto-btn-text">Text</button>
<button class="picto-btn picto-btn-icon">
  <svg><!-- icon --></svg>
</button>
```

## Disabled

```html
<button class="picto-btn picto-btn-filled" disabled>Disabled</button>
```

All buttons are fully themeable and responsive, using Pittorica design tokens.
