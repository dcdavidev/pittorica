# pittorica

Pittorica is a painterly, CSS-first UI framework that brings elegant, lightweight structure to modern interfaces.

## Table of Contents

- [Features](#features)
- [Documentation](#documentation)
- [License](#license)

## Features

- **Design tokens** for colors, spacing, typography, radius, elevation, and motion
- **Responsive**: all scales use `rem` and `clamp()` for fluid layouts
- **Dynamic color palettes** with `color-mix()` and easy theming
- **Utility classes** for rapid prototyping (spacing, layout, colors, typography, etc.)
- **Base CSS components** for buttons, inputs, cards, and more
- **SSR-friendly**: zero-runtime, pure CSS, works with any framework
- **Ready for React wrappers** and TypeScript support
