# Stack

Flexbox-based vertical or horizontal stack for spacing between items.

- `.picto-stack-horizontal` for horizontal layout

## Usage

```html
<div class="picto-stack">
  <div>Item 1</div>
  <div>Item 2</div>
</div>
<div class="picto-stack picto-stack-horizontal">
  <div>Item 1</div>
  <div>Item 2</div>
</div>
```
